# CALCULADORA DE COMPRAS.
simula o calculo de pagamento em uma loja onde: 
    - Avista no dinheiro, 10% de desconto.
    - Avista no cartão, 5% de desconto.
    - 2X no cartão, preço normal.
    - Acima de 3X no cartão, 20% de juros.
